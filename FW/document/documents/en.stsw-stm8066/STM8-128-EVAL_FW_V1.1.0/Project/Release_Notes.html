STM8-128-EVAL update history

 

V1.1.0 / 11-March-2011

Main Changes

  • Update overall demonstration FW to use the STM8S_EVAL drivers
  • functions.c/.h
    • Remove "mono_lcd.h" file inclusion and replace it by #include "stm8s_eval.h" and #include "stm8s_eval_lcd.h"
    • Remove LEDs definition and macros for Leds handling and replace them by the stm8s_eval drivers functions
    • Remove Init_SPISimplex() function to be handled by stm8s_eval drivers functions
    • Remove Init_SPIFullDuplex() function to be handled by stm8s_eval drivers functions
    • Remove unused functions putchar(), InitRAM1(), Init_TIM3() and Init_TIM4()
    • Update Init_GPIO() function to use tm8s_eval drivers functions
    • Remove unused defines:
      • RAM1_Start
      • RAM1_END
      • APPLICATION_STATUS_1_EXT_IT
      • APPLICATION_STATUS_1_RTCAWU_IT
      • APPLICATION_STATUS_1_PVD_IT
      • APPLICATION_STATUS_1_PVD_LEVEL
      • APPLICATION_STATUS_1_NAVIGATION_ENABLED
      • APPLICATION_STATUS_1_LED_ACTIVITY_ENABLED
      • HELP_DISPLAY_STATUS_ENABLE
  • demo_microsd.c/.h
    • Update to use stm8s_eval drivers function and specific "stm8s_eval_spi_sd" driver and "stm8s_eval_lcd" driver
  • main.c/.h
    • Update MelodyCallBack() function to use "stm8s_eval_lcd" driver functions
  • joystick_button.c/.h
    • Update ReadJoystick() function to use stm8s_eval drivers functions
    • Remove ReadButton() function 
    • Remove BUTTON_ID_T typedef and JOYSTICK_ID_T typedef because handled by stm8s_eval drivers
    • Remove unused defines :
      • BUTTON_STATUS_JOYSTICK_DEBOUNCE_ENABLE
      • BUTTON_STATUS_BUTTON_DEBOUNCE_ENABLE
  • menu.c/.h
    • Update MenuUpdate() function to use stm8s_eval drivers functions
  • demo_about.c/.h
    • Update ABOUT_Exec() function to use "stm8s_eval_lcd" driver functions
  • demo_help.c/.h
    • Remove the following defies
      • HELP_LINE1_TEXT (" LEFT or RIGHT\nto scroll menu ")
      • HELP_LINE2_TEXT (" Press SEL to \nenter sub-menu ")
      • HELP_LINE3_TEXT ("   Move UP to \n exit sub-menu ")
    • Update HELP_Exec() function to use "stm8s_eval_lcd" driver functions
  • demo_beeper.c/.h
    • Update BEEPERDEMO_Reset() function to use stm8s_eval drivers functions
    • Update BEEPERDEMO_Exec() function to use "stm8s_eval_lcd" driver functions
  • demo_system.c/.h
    • Update SYSTEM_Exec() function to use stm8s_eval drivers functions and "stm8s_eval_lcd" driver functions
  • demo_adc.c./h
    • Update Bar_16_16, Bar_15_16, Bar_14_16, Bar_13_16, Bar_12_16, Bar_11_16, Bar_10_16, Bar_9_16, Bar_8_16, Bar_7_16, Bar_6_16, Bar_5_16, Bar_4_16, Bar_3_16, Bar_2_16 and Bar_1_16 with hexadecimal definition instead of binary(not supported by IAR compiler)
    • Update ADC2DEMO_Reset() and ADC2_DisplayOnLeds() functions to use stm8s_eval drivers functions
    • Update the following functions to use stm8s_eval drivers and "stm8s_eval_lcd" driver functions:
      • ADC2DEMO1_Init()
      • ADC2DEMO2_Init()
      • ADC2DEMO3_Init()
      • ADC2DEMO_Exec()
      • ADC2DEMO3_Exec()
    • Add static functions DisplayCGRAM0() and DisplayCGRAM1()
  • demo_timer.c/.h
    • Update TIMERDEMO_Reset() function to use stm8s_eval drivers functions
    • Update TIMERDEMO_Exec() function to use stm8s_eval drivers and "stm8s_eval_lcd" driver functions
    • Update TimerDisplay() function to use "stm8s_eval_lcd" driver functions
  • demo_pwm.c/.h
    • Update to use stm8s_eval drivers and "stm8s_eval_lcd" driver functions
  • demo_games.c.h
    • Update to use stm8s_eval drivers and "stm8s_eval_lcd" driver functions
  • buzzer.c/.h
    • Update PlayMelody() function to use stm8s_eval drivers functions.
